About the Team

We built MacroTrack as a simple, responsive food planner to practice collaboration, Git workflows, and clean frontend code.

Team Member 1

Role: Frontend / HTML

  • Built page layout and Bootstrap structure
  • Created navigation and responsive design
  • Worked on index.html and about.html

Team Member 2

Role: JavaScript / Logic

  • Implemented food entry logic and rendering
  • Added localStorage persistence
  • Improved event handling and UI feedback

Team Member 3

Role: Styling / UI

  • Designed styles.css (cards, spacing, inputs)
  • Implemented dark mode styling support
  • Polished overall UI consistency

How We Collaborated

We used feature branches for development and merged changes into main using pull requests. Each PR was reviewed by a teammate to ensure code quality and prevent breaking changes.

We also created and resolved a merge conflict intentionally to demonstrate real-world collaboration and Git conflict resolution.
